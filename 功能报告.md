### 3. 💬 RAG 问答引擎

#### 检索增强生成（RAG）流程

```
用户提问："list是什么"
        │
        ▼
  ① TF-IDF 检索
  ┌──────────────────────────┐
  │ TfidfVectorizer          │
  │ analyzer='char_wb'       │  ← 中文逐字分词
  │ ngram_range=(1, 3)       │  ← 1~3字组合匹配
  │ + 余弦相似度排序           │
  └──────────────────────────┘
        │
        ▼ Top-3 最相关文档
  ② 拼接提示词
  ┌──────────────────────────┐
  │ 系统指令                  │
  │ + 参考资料（检索结果）      │
  │ + 用户问题                │
  └──────────────────────────┘
        │
        ▼
  ③ DeepSeek 生成
  ┌──────────────────────────┐
  │ model: deepseek-chat     │
  │ temperature: 0.3（严谨）   │
  │ max_tokens: 2048         │
  └──────────────────────────┘
        │
        ▼
  AI 回答 + 📖 参考片段展示
```

#### 关键参数

| 参数 | 值 | 说明 |
| --- | --- | --- |
| 检索算法 | TF-IDF + 余弦相似度 | 纯 Python，兼容性好 |
| 检索数量 | Top-3 | 返回最相关的3条知识 |
| AI 模型 | deepseek-chat | DeepSeek V3 |
| 温度 | 0.3 | 严谨模式，减少编造 |
| 最大长度 | 2048 tokens | 足够详细回答 |

### 4. 🧠 对话记忆

```
第1轮                           第2轮
┌──────────────────┐           ┌──────────────────┐
│ User: 指针是什么   │           │ User: 它和引用区别  │
│ AI: 指针是...      │    ──→    │                   │
└──────────────────┘           │ 发送给 DeepSeek：  │
                                │ [U1, A1, U2+RAG] │
                                │ AI 知道"它"=指针  │
                                └──────────────────┘
```

| 特性 | 说明 |
| --- | --- |
| 上下文理解 | AI 能看到完整对话历史，理解"它"、"这个"等代词指代 |
| 独立清空 | "清空对话"按钮只清对话，不删知识库 |
| 切换清空 | 切换知识库时自动清空对话（避免张冠李戴） |

### 5. ⚡ 性能优化

| 技术 | 作用 |
| --- | --- |
| `@st.cache_resource` | TF-IDF 计算结果缓存，相同知识库不重复计算 |
| `@st.cache_resource` | DeepSeek 客户端单例，避免重复创建连接 |
| `st.session_state` | 知识库文本持久化，只存原文不存计算对象 |
