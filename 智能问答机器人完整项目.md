## 🧠 RAG 原理

TEXT

```
RAG = Retrieval-Augmented Generation = 检索增强生成

用户问："malloc怎么用？"
        │
        ▼
  ① 检索（Retrieval）
  在知识库里搜索相关内容
  用 TF-IDF 把文本转成向量
  计算余弦相似度
  找到最相关的3条知识
        │
        ▼
  ② 增强（Augmented）
  拼接提示词 = 知识 + 规则 + 问题
        │
        ▼
  ③ 生成（Generation）
  发送给 DeepSeek
  DeepSeek 严格基于给的文档回答
  不会胡编乱造
```

### TF-IDF 替代 chromadb 的原因

||chromadb|scikit-learn|
|---|---|---|
|兼容性|Python 3.14 报错|纯 Python，任何版本可用|
|安装|需要 C++ 编译工具|`pip install` 直接搞定|
|效果|语义搜索更好|关键词匹配，RAG 场景够用|
|部署|Streamlit Cloud 不兼容|完美兼容|

## 📌 关键技术要点

### DeepSeek API 调用模板

PYTHON

```
from openai import OpenAI

client = OpenAI(
    api_key="sk-xxx",
    base_url="https://api.deepseek.com",
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[
        {"role": "system", "content": "你是..."},
        {"role": "user", "content": "问题"}
    ],
    temperature=0.7,
    max_tokens=2048,  # DeepSeek 必须设置
)

answer = response.choices[0].message.content

```
